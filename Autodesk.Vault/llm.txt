## Overview

Type-safe C# SDK for the Autodesk Vault Data REST API (v2). Provides two access patterns:

1. **Fluent URL API** (`client.Api.*`) — mirrors the REST path structure exactly
2. **Manager API** (`client.{Manager}`) — high-level methods with strongly-typed params and XML doc comments

Unlike cloud-based APS SDKs, this SDK connects to on-premise Vault servers via a configurable `vaultServerUrl`. The base URL is `{vaultServerUrl}/AutodeskDM/Services/api/vault/v2`.

## Quick Start

```csharp
using Autodesk.Vault;

// 2-legged (server-to-server with user impersonation)
var client = new VaultClient(
    () => Task.FromResult("YOUR_ACCESS_TOKEN"),
    vaultServerUrl: "https://your-vault-server",
    userId: "user@company.com");

// 3-legged (user context)
var client = new VaultClient(
    () => Task.FromResult("YOUR_3LEGGED_TOKEN"),
    vaultServerUrl: "https://your-vault-server");

// Manager approach (recommended)
var vaults = await client.Informational.GetVaultsAsync();

// Fluent URL approach — direct REST mapping
var vaults = await client.Api.Vaults.GetAsync();
```

## Constructors

```csharp
// 2-legged: server-to-server with user impersonation
public VaultClient(Func<Task<string>> get2LeggedAccessToken, string vaultServerUrl, string userId, HttpClient? httpClient = null)

// 3-legged: user context
public VaultClient(Func<Task<string>> get3LeggedAccessToken, string vaultServerUrl, HttpClient? httpClient = null)
```

- `get2LeggedAccessToken` / `get3LeggedAccessToken`: async function returning a valid OAuth bearer token
- `vaultServerUrl`: valid absolute HTTP/HTTPS URL to the Vault server (e.g. `"https://your-vault-server"`, `"http://10.148.0.1"`)
- `userId`: (2-legged only) email address of the Vault user to impersonate
- `httpClient`: optional custom HttpClient (default includes retry + rate-limit handling)

## Built-in Middleware

- **Rate limiting**: auto-retries on 429 with Retry-After / exponential backoff
- **Error handling**: throws `HttpRequestException` with status code and full response in `ex.Data["context"]`
- Disable per-request: pass `new RequestConfiguration<T> { Options = { new ErrorHandlerOption { Enabled = false } } }`
