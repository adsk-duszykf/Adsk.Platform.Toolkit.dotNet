## Overview

Type-safe C# SDK for the Autodesk Tandem Digital Twin REST APIs. Provides two access patterns:

1. **Fluent URL API** (`client.Api.*`) — mirrors the REST path structure exactly
2. **Manager API** (`client.*Manager`) — high-level methods with strongly-typed params and XML doc with links to official docs

## Quick Start

```csharp
using Autodesk.Tandem;

var client = new TandemClient(() => Task.FromResult("YOUR_ACCESS_TOKEN"));

// Manager approach (recommended)
var groups = await client.GroupsManager.GetGroupsAsync();

// Fluent URL approach — direct REST mapping
var response = await client.Api.Tandem.V1.Groups.GetAsync();
```

## Constructor

```csharp
public TandemClient(Func<Task<string>> getAccessToken, HttpClient? httpClient = null)
```

- `getAccessToken`: async function returning a valid OAuth bearer token
- `httpClient`: optional custom HttpClient (default includes retry + rate-limit handling)

## Built-in Middleware

- **Rate limiting**: auto-retries on 429 with Retry-After / exponential backoff
- **Error handling**: throws `HttpRequestException` with status code and full response in `ex.Data["context"]`
- Disable per-request: pass `new RequestConfiguration<T> { Options = { new ErrorHandlerOption { Enabled = false } } }`
